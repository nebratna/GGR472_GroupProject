Key Terms:

What is NDVI?
And how does it measure greenness?

Normalized Difference Vegetation Index (NDVI) is an incredible metric to measure how green an area is using satellite imagery.

Learn More
What is LST?
And why does it matter?

Land Surface Temperature (LST) is a way of measuring how warm an area gets, also using satellite imagery.

Using the map:

Explore NDVI and LST in Toronto's Neighbourhoods

Use the toggles and checkboxes on the panel on the left in the main map to see how NDVI and LST values vary for Toronto's neighbourhoods, and to see which neighbourhoods fall into specific ranges. You can choose to turn legends ON and OFF. If there is an overlap between the red (LST) and green (NDVI) layers, the resulting neighbourhoods are varying shades of orange and brown. You can also click on each neighbourhood to see its name and the corresponding LST and NDVI values (if you have both layers turned on, the popup boxes will appear as two separate boxes, one directly underneath the other. Close the first popup to see the second popup under it.)
The "compare with slider" button allows you to compare the NDVI and LST maps with a quick swipe. You can return to the main map to find the exact values for both parameters.
